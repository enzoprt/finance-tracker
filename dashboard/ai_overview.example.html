AI Overview — written by Claude

This tab is not computed by a script like the rest of the dashboard - it's a hand-written analysis of your real financial situation (net worth composition, portfolio construction, performance vs benchmark, tax-wrapper usage, opinions on what to consider changing), produced by asking Claude to read data/summary.json plus your real broker/account structure and write it up.

The two real content files (dashboard/ai_overview.en.html and dashboard/ai_overview.fr.html) are gitignored because they contain real personal financial data once written - this ai_overview.example.html file is the generic placeholder that ships in the repo instead, so the project structure makes sense without leaking anyone's actual numbers.

How to generate your own

Run python -m scripts.build_summary at least once, then ask a Claude Code session (in this project) to read data/summary.json, your real account structure (Saxo sub-accounts, PEA eligibility, fund overlap, etc.), and write the same analysis twice - into dashboard/ai_overview.en.html and dashboard/ai_overview.fr.html - using the same card/table markup as the rest of this dashboard (see dashboard/template.html for the CSS classes available: .card, .card-head, table, .hero-sub, etc.). The EN/FR toggle button on the AI Overview tab just show/hides #ai-overview-en vs #ai-overview-fr - both are always embedded in the page, no extra language means extra build logic beyond adding a third file and a third toggle button. Neither file auto-updates - re-ask periodically after meaningful changes.
